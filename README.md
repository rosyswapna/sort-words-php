# Sort words

Sort words in the order of sum of position number of each alphabet, like A => 1, B=> 2 ....Z => 26

## INPUT
$words = array(
    "GMAIL",
    "YAHOO",
    "OUTLOOK",
    "HUBSPOT",
    "AOL",
    "YANDEX",
    "GMX");
 ## OUTPUT
 Array
(
    [2] => OUTLOOK
    [3] => HUBSPOT
    [5] => YANDEX
    [1] => YAHOO
    [6] => GMX
    [0] => GMAIL
    [4] => AOL
)
